# Teste RobotEasy

## Descrição
App Spring + VueJs para consulta de CEP
App em docker

## Como Executar

### Requisitos
- Docker
- Docker Compose

### Portas
- Front-end: 80
- Back-end: 8080

### Configuração
1. Clone o repositório:
    ```bash
    git clone git@github.com:igorpopovicz/robotEasy.git
    cd robotEasy
    ```

2. Rode com o docker compose:
    ```bash
    docker compose up -d --build
    ```
3. Acesse no web: http://localhost
